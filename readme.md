### generator  

###### Description:  
The pulse generator, implemented on STM32, can generate:  
1. single pulse;  
2. continuous pulse generation;  
3. generation of pulse bursts.  

Init mode:
- width - 1 us;  
- period - 10 us;  
- pack period - 100 us;  
- number of impulses - 5.  

Max number of impulses in package - 2048.  
Min pulse width 100 ns.
